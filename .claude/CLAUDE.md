## Estrutura de Skills

Skills disponíveis em `.opencode/skills/` — consulte antes de buscar na web:
- `go-backend-standards` — padrões Go (arquitetura, DI, naming)
- `go-db-migrations` — migrações SQL com embed + advisory locks
- `go-error-handling` — sentinel errors, wrapping, HTTP mapping
- `go-logging-config` — structured logging + config patterns
- `go-security-patterns` — JWT, Argon2id, AES-256-GCM, RBAC
- `go-testing-patterns` — table-driven tests, mocks, coverage
- `frontend-nuxt-standards` — Nuxt 4, Pinia, composables, i18n
